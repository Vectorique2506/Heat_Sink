# Heat_Sink
A physics-based real-time thermal simulator that reads live CPU  power consumption and predicts the complete 2D temperature  distribution across a CPU heat sink using the finite difference  method.


# Physics Principles Used
- Fourier's Law of Heat Conduction
- Newton's Law of Cooling  
- 2D Steady State Heat Equation
- Fin Efficiency Theory
- Thermal Resistance

## Features
- Live CPU power and temperature reading via hardware sensors
- Real-time 2D heatmap of temperature distribution
- Interactive controls — material, fin geometry, fan speed, power
- Fin geometry comparison (rectangular, triangular, pin fins)
